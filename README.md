# Loan Prediction Using Machine Learning

This project predicts whether a loan applicant is eligible for loan approval using supervised machine learning techniques.

## Models Used
- Decision Tree Classifier
- Support Vector Classifier (SVC)
- K-Nearest Neighbors (KNN)

## Workflow
- Data preprocessing and missing value handling
- Encoding categorical variables
- Train-test split
- Model training and evaluation using accuracy

## Result
The Decision Tree model achieved the highest accuracy among the tested algorithms.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
